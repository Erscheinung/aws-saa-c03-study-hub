⚖ ALB vs NLB NETWORK
ALB (Application)
  • Layer 7 (HTTP/HTTPS)
  • Path-based routing
  • Host-based routing
  • WebSocket support
  • WAF integration
NLB (Network)
  • Layer 4 (TCP/UDP)
  • Ultra-low latency
  • Static IP support
  • Preserves source IP
  • Millions of requests/s
When to use which?
"Need static IP or extreme performance? NLB. Need content-based routing? ALB."
🛡 Security Groups vs NACLs SECURITY
Security Groups
  • Instance level
  • STATEFUL
  • Allow rules only
  • All rules evaluated
  • First line of defense
NACLs
  • Subnet level
  • STATELESS
  • Allow + Deny rules
  • Rules processed in order
  • Backup line of defense
Key: "Block specific IP" = NACL (deny rule). Security Groups can only allow.
⚡ Redis vs Memcached DATABASE
Redis
  • Data persistence
  • Replication & HA
  • Complex data types
  • Pub/Sub messaging
  • Sorted sets, lists
Memcached
  • No persistence
  • Multi-threaded
  • Simple key-value
  • Simpler architecture
  • Pure caching
